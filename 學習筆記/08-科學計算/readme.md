# 科學計算
計算科學，又稱科學計算，是一個與數學模型構建、定量分析方法以及利用電腦來分析和解決科學問題相關的研究領域。
在實際應用中，計算科學主要應用於：對各個科學學科中的問題，進行電腦類比和其他形式的計算。

這一領域不同於電腦科學（對於計算、電腦以及資訊處理的研究），同時也異於科學和工程學的傳統形式——理論與實驗。
科學計算技術要想獲得理解，主要需要通過在電腦上實現的數學模型進行分析。
